Tablas de multiplicar

Codigo para while:
while (multiplicador<=10){
imprimir(tablaUsuario + " " + " x " + multiplicador + " = "
+(tablaUsuario*multiplicador));
multiplicador=multiplicador+1;
}

Codigo para for:
for(var multiplicador = parseInt(1);multiplicador<=10;multiplicador=multiplicador+1){
imprimir(tablaUsuario + " " + " x " + multiplicador + " = "
+(tablaUsuario*multiplicador));
}

Se puede ahorrar el +1 del codigo con declarar la var multiplicador++

Ejemplo: var contador = 1
while(contador <= 10) {
imprimir("Contador actual: " + contador);
contador++;
}
